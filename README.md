# 🚀 สร้าง Frontend (Vite) สำหรับทดสอบ UI

## ติดตั้งและสร้างโปรเจกต์
```bash
# 1. สร้างโปรเจกต์ Vite (ตั้งชื่อว่า mock-frontend)
npm create vite@latest mock-frontend -- --template react

# 2. เข้าไปในโฟลเดอร์โปรเจกต์
cd mock-frontend

# 3. ติดตั้ง Library พื้นฐาน
npm install

# 4. ทดลองรันเซิร์ฟเวอร์
npm run dev
```
